# BM Program

## Getting started

#### Requirement

This app uses node v15.4.0, or npm v15.4.0

#### Project setup

Install project dependencies you can use npm:

```
yarn install
```

#### Run development server

```
yarn start
```

#### Run linters

```
npm run lint
```

#### Run tests

```
npm test
```

#### Create production build

```
npm run build
```
